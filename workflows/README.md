# My n8n Workflows

This folder is where you will save your automation workflows. 

**How to use this folder:**
1. Open your completed workflow in n8n.
2. Click the menu in the top right corner and select **Download**.
3. Click **Add file** -> **Upload files** in this GitHub folder and upload the `.json` file you just downloaded.
4. Make sure to name it clearly (for example: `week3.json`).
